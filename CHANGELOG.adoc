== {compare-url}/v0.14.1\...HEAD[Unreleased]

=== Added

* Support `no_std` mode
* Add EPS rendering support
* Add rMQR code encoder
* Support automatic determines the smallest Micro QR code
* Re-export `image` crate ({pull-request-url}/13[#13])

=== Changed

* Improve rendered PIC quality
* Change XML declaration of SVG
* Reduce size of SVG output
* Change MSRV to 1.85.0
* Change behavior of `Version::is_micro` to return `false` if the parameter is
  out of the valid range
* Rename crate to `qrcode2`
* Make the `Error` trait available in `no_std` mode ({pull-request-url}/14[#14])

=== Removed

* Remove binary target
* Remove `bench` feature
* Remove deprecated methods ({pull-request-url}/3[#3])
